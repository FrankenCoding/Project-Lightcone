# Project-Lightcone

Prototype for a card-based roguelike about building AGI. See [design_doc](docs/design_doc.md) for the full plan.

This repository contains the initial Godot project under `runaway-stack/`.

## Running the demo

1. Open Godot 4 and load the `runaway-stack` project folder.
2. Run the `Main` scene. The console will print the cards drawn at the start of the turn.
3. Add your own art in `runaway-stack/art/cards` and update `data/cards.yaml`.
