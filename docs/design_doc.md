# Runaway Stack – Design Document (v0.2)

## 0. Elevator Pitch

A single‑player **roguelite deck‑builder** where you steer an AI lab from the modern deep‑learning boom (AlphaGo, 2016) toward the first *safe* AGI, balancing **Compute ⚡**, **Insight ✨**, and **Alignment Risk ☢**. A post‑launch *Singularity* expansion pushes beyond AGI and lets players ride—or restrain—the intelligence explosion.

---

## 1. Theme & Timeline

| Phase | Years Covered | Narrative Goal | New Systems |
| --- | --- | --- | --- |
| **Phase 1 – AGI Race** | 2016‑2045 | Build the first aligned AGI before Risk ≥ 10 | Core resources, deck‑builder loop |
| **Phase 2 – Singularity** | 2046‑∞ (optional after winning Phase 1) | Navigate runaway take‑off, decide whether to merge, constrain, or transcend | Exponential resource growth, *ASI Dilemmas*, legacy cards evolve |

*All scientists, labs, and chips are thin parodies (e.g. **Offrey Hinton**, **StarGate‑1 GPU**) for legal safety.*

> **Legacy Relics.** Rare, pre‑2010 artifacts such as *"Turing Bombe Replica"* or *"Perceptron Punch Card."* They **take up a relic slot** but add unique combos—for example, *Bombe Replica* counts as both Hardware **and** Algorithm, enabling early "Trinity" plays even before you draw modern cards.

---

## 2. Core Resources & Turn Structure

| Resource | Generated By | Spent On | Failure State |
| --- | --- | --- | --- |
| **Compute ⚡** | Hardware cards, Infra events | Algorithms, Training actions | Shortage ⇒ costs ×1.5 for the turn |
| **Insight ✨** | Researcher upkeep, Breakthrough events | Upgrades, bonus draws | Hoarding ≥ 20 ⇒ +1 Risk per turn |
| **Alignment Risk ☢** | Breakthroughs, certain synergies | — | **Risk ≥ 10 ⇒ Rogue AI ⇒ Run ends** |

### Core Gameplay Loop (one "Year" ≈ 1 Turn)

1. **Draw Phase** – Draw 5 cards (hand limit 7).
2. **Action Phase** – Spend up to **3 Actions** and any available ⚡. Typical actions:

   * *Play* a card (Researchers, Hardware become *Persistent*; others resolve and go to discard).
   * *Train* (convert ✨ to AGI at 1 ✨ → 1 AGI, max once/turn).
   * *Upgrade* (pay ✨ to bump a card’s tier, exhausts the card this turn).
3. **Maintenance** – Gain passive ✨ from Researchers and ⚡ from Hardware; resolve start/end‑of‑year effects; check Risk.
4. **Era Progression** – Every 5 turns the map advances one node (Conference, Vendor, Policy, Boss, etc.).

---

## 3. Card Families

1. **Researchers** – Persist; +✨ each Maintenance.
2. **Algorithms** – Burst AGI, high ⚡ cost.
3. **Hardware** – Persist; +⚡, often +☢ side‑effects.
4. **Datasets** – Deck/thinning and draw manipulation.
5. **Governance / Ethics** – Lower ☢, gate combos.
6. **Events** – Powerful one‑shots with pros/cons.
7. **Artifacts (Relics)** – Global passives (max 3, unless upgraded). Includes **Legacy Relics**.

---

## 4. Combo System (Balatro‑flavored)

| Combo | Recipe | Payoff |
| --- | --- | --- |
| **Trinity** | Play Researcher + Algorithm + Hardware in a single Action Phase | ×3 ✨ immediately |
| **Paper Blitz** | Have 5 unique Researchers in play at turn end | +10 AGI, reshuffle discard into deck |
| **GPU Overclock** | Spend ≥ 8 ⚡ in one Action | +15 AGI *and* gain a **Faulty Component** curse |

---

## 5. Roguelite Map & Bosses

* **Nodes:** *Conference* → draft, *Vendor* → shop, *Policy* → risk/reward, *Mystery* → RNG events.
* **Era Bosses (one per Era):**

  1. *Regulatory Takeover* – forces Risk trade‑offs.
  2. *Open‑Source Fork Gone Wild* – spawns duplicate Algorithm curses.
  3. *Rival Megalab* – steals Compute unless stunned each turn.

Post‑AGI Singularity mode introduces the **ASI Oversight Council** mega‑boss.

## 6. Art & Audio Style

* Visual references: *Slay the Spire* layout, *Hearthstone Battlegrounds* polish, plus anime‑inflected character art. **FTL is another game to reference**
* Two candidate palettes—will A/B test during prototype:

  1. **Glitch Terminal** – neon greens & magentas on charcoal.
  2. **Solar-Ink** – muted teals & warm oranges (anime poster vibe).
* Audio layers: crunchy key-clicks, server-fan hums; Risk ≥ 7 triggers subtle klaxons and rising tape-saturation hiss.

## 7. Tech & Platform Plan

### 7.1 Engine & Languages

* **Engine:** **Godot 4.3** (MIT-licensed, lightweight, great 2-D & WebGL export).
* **Languages:**

  * **GDScript** for most gameplay scripts—Python-like, rapid iteration.
  * **C#** (optional) for performance-critical systems (e.g. card shuffler, save encryption).
  * **YAML/JSON** for data-driven card definitions.
* **Target Platforms:** Win / macOS / Linux desktop; WebGL prototype; iOS/Android post-beta.

### 7.2 Repo & File Layout (monorepo)

```
runaway-stack/
 ├─ addons/              # Godot plugins (YAML loader, shaders)
 ├─ art/
 │   ├─ cards/           # 512×768 PNG art, snake_case names
 │   └─ ui/              # Frame sprites, icons, FX atlases
 ├─ data/
 │   ├─ cards.yaml       # Card stats + art filename
 │   └─ relics.yaml
 ├─ scenes/
 │   ├─ main.tscn        # Root viewport & game manager
 │   ├─ ui/              # HUD, pop-ups
 │   └─ cards/           # Card scene (generic), AnimPlayer, signals
 ├─ scripts/
 │   ├─ Card.gd          # Instanced by DeckManager
 │   ├─ DeckManager.gd   # Draw/discard/shuffle logic
 │   ├─ TurnManager.gd   # Phases, actions-left, end-turn hooks
 │   └─ SaveSystem.cs    # Cross-platform JSON save, gzip-compressed
 ├─ shaders/             # Glitch bloom, hologram pulse
 └─ export_presets.cfg   # Godot export targets
```

### 7.3 Asset Pipeline (Art → In-game)

1. **Concept** in image gen → pick style.
2. **Photoshop/Paint over**: final 512×768 PNG, transparent where needed.
3. Name file `sam_aitman_cyber.png`, drop in `art/cards/`.
4. Add entry in `data/cards.yaml`:

   ```yaml
   - id: sam_aitman
     name: "Sam Aitman"
     type: researcher
     insight_per_turn: 1
     art: "sam_aitman_cyber.png"
     cost: 2
     risk_on_play: 0
   ```
5. On game start the **YamlLoader** plugin converts YAML → JSON → Godot **Dictionary**; **DeckManager** spawns a `Card` scene, sets sprite to `art/cards/<art>` and fills UI labels.

### 7.4 UI Frame & Stat Binding

* Card scene (`Card.tscn`) hierarchy:

```
Card
├─ FrameSprite       # 9-sliced SVG, theme variant
├─ ArtSprite         # 512×768 texture
├─ CostLabel         # Top-left ⚡ cost
├─ RiskIcon          # Top-right ☢ for risky plays
├─ NameLabel         # Banner text
├─ TextArea          # Description (BBCode-rich)
├─ RarityStar        # Bottom-left
└─ ValueLabel        # Bottom-right (Insight/AGI)
```
- Dynamic theme: `ThemeManager` swaps **Glitch** vs **Solar-Ink** frames via `ThemeData.tres`.

### 7.5 Build & CI
- **Git + Git-LFS** for binary assets.
- **GitHub Actions**: on push → headless Godot export → itch.io butler push (dev branch) + WebGL artifact.
- **Pre-commit hooks**: YAML lint, GDScript format, PNG size guard (<300 KB).

> **Good division of labor**
> *You*: generate/paint art, update `cards.yaml`, run local Godot editor, tweak UI colors.
> *Me*: maintain scripts, loaders, CI pipeline, and performance profiling.

### 7.6 Stretch-Stack Ideas
- **Lua hot-reload** for modding (via godot-lua-plugin).
- **Rust GDNative** micro-optimizations if deck size & combos explode.
- **Firebase** real-time DB for future PvP ladder.
